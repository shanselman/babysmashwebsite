# BabySmash! - Modern Website

A complete reimagining of the BabySmash! website with a fresh, modern design built using:

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with gradients, animations, and responsive design
- **Data-star.dev** - Lightweight reactivity framework for interactive features
- **No Backend** - Pure client-side static website

## How to Use

Simply open `index.html` in a web browser or serve via any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000/` in your browser.
